Exploring dsL with dplyr

The data stage dsL at which weve just arrived, is still a bit cumbersome to handle and to think about. Ultimately, we want to comment on the relationship of just a few variables, so lets start with the bare minimum.

dplyr::filter()

We know that dsL contains data for

The function filter() of the dplyr package allows to select rows that match the given criteria. Lets select all the rows, in which the column PairID contains the numeric value 28 

dsM <- dsL %>%
  dplyr::select (Case, PairID, TwinIDF, ZygosityF, time, CompAge, epi_e, epi_n, DemEver) %>%
  dplyr::filter (PairID == 28)
